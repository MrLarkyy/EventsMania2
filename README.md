# EventsMania

Welcome to EventsMania! This is a robust event management plugin designed for modern Minecraft servers. It allows you to
create engaging, time-limited challenges for your players with ease.

## Features

* **Dynamic Event System:** Manage various types of in-game events with dedicated handlers and managers.
* **Leaderboards:** Track player performance and competition in real-time.
* **Flexible Configuration:** Fully customize plugin behavior, event details, and localized messages via specialized
  YAML files.
* **Data Persistence:** Reliable data handling and serialization for player statistics and event states.

## Requirements

* **Platform:** Minecraft Server (Paper)
* **Java Version:** 21
* **API Version:** 1.21

## Integration & Dependencies

EventsMania is designed to work seamlessly with several popular plugins to enhance functionality:

* **Waves:** (Required) Core dependency for plugin operation.
* **PlaceholderAPI:** Optional support for custom placeholders in messages and leaderboards.

## Configuration

The plugin utilizes several configuration files located in the `resources` directory:

* `config.yml`: Main plugin settings.
* `events.yml`: Definitions and parameters for your custom events.
* `messages.yml`: Customizable strings for all player-facing communications.
